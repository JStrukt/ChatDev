# Simpsons Tamagotchi Game

A fun and interactive game where you can take care of your favorite Simpsons character!

## Quick Install

This game is developed in Java. Ensure you have Java Development Kit (JDK) installed on your machine. If not, you can download it from [here](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).

## ðŸ¤” What is this?

This is a Tamagotchi-style game based on the popular TV show, The Simpsons. You can choose your pet from Bart, Lisa, Maggie, Homer, or Marge Simpson. Your task is to take care of your chosen character by feeding, playing with, and cleaning them. Be careful, if your character's health, happiness, hunger, or cleanliness reach critical levels, the game will end!

## ðŸŽ® How to Play

1. Run the main.java file to start the game. You will be greeted with a welcome message and asked to choose your character.

2. Enter the name of your chosen character. You can choose from Bart, Lisa, Maggie, Homer, or Marge.

3. Once you've chosen your character, you can perform actions to take care of them. Enter "feed" to feed your character, "play" to play with them, or "clean" to clean them.

4. Keep an eye on your character's health, happiness, hunger, and cleanliness levels. If any of these reach a critical level, the game will end!

Please see the comments in the code for a detailed explanation of each class and method.

- character.java: This class represents a character from the Simpsons. It has attributes for the character's name, health, happiness, hunger, and cleanliness. It also has methods to feed, play with, and clean the character.

- game.java: This class represents the game itself. It has methods to start the game, choose a character, and perform actions on the character.

- main.java: This class is the entry point of the application. It creates an instance of the Game class and starts the game.
