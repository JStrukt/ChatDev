# Snake Game

A classic Snake game developed in Python using Pygame.

## Quick Install

Before you can run the game, you need to install the required dependencies.

If you have pip installed, you can install the dependencies by running:

`pip install -r requirements.txt`

If you're using conda, you can create a new environment and install the dependencies with:

`conda create --name <env> --file requirements.txt`
